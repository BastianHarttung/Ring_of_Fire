# Ring Of Fire

This is a multiplayer drinking-game famous in America.

### Start
1. Get the Players
2. Get your Drinks
3. Hit the Start Button

![img.png](src/assets/img/readme/Start.png)

### Game
1. First Add Players
2. Then Start the Game by clicking on the Card Stack
